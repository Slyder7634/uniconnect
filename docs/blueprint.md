# **App Name**: UniBridge

## Core Features:

- Secure User Authentication: Enable secure login for students and teachers using unique IDs. Create random fake credentials to be used for testing during the demo phase.
- Personalized Schedules: Display personalized course schedules for each student.
- Grade Viewing: Allow students to view their grades for completed courses.
- University Announcements: Provide a central hub for university-wide announcements.
- Academic Resources: Create access point academic resources such as library links, research databases, or writing support.

## Style Guidelines:

- Primary color: Navy blue (#2E5B98) to convey trust and professionalism, suitable for a university setting.
- Background color: Light gray (#F0F4F8), offering a clean and modern look, minimizing distraction.
- Accent color: Soft gold (#A9927D) to provide a touch of university branding.
- Font: 'Inter', a sans-serif font that is modern, readable, and suitable for both headlines and body text.
- Use a consistent set of simple, professional icons, aligned with the university's branding guidelines. 
- Employ a clean, responsive layout optimized for various screen sizes, ensuring easy navigation and content discovery.
- Incorporate subtle transitions and animations for a polished and engaging user experience. Focus on loading screens.
